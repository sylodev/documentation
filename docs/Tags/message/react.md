description: Reacts to the message in context with [emoji]. 
---
# {message.react;&lt;emoji>;[message id];[message channel]}
## Examples
{% tabs %}
{% tab title="Input" %}
```text
{message.react;eggplant}
```
{% endtab %}
{% tab title="Output" %}
```text

```
{% hint style="info" %}
The message in context would have the eggplant emoji added to it.
{% endhint %}
